A simple Flutter-based hospital appointment booking application.
This project allows users to view doctors, check available slots, and book appointments without using any database (in-memory storage only).

Features

View list of doctors

Book appointment (with date & slot selection)

Prevents double booking

View & cancel booked appointments

About Hospital page

Asset images supported (logo & doctor images)

Tech Stack

Flutter (Dart)

Material UI

In-memory data models (no backend)
